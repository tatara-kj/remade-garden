# Struktura przyszłego poradnika CarLux

Katalog jest przygotowany pod późniejsze podstrony SEO. Artykuły nie są jeszcze dodawane do nawigacji ani mapy witryny.

Rekomendowane adresy:

- `/poradnik/jednoetapowa-korekta-lakieru.html`
- `/poradnik/powloka-ceramiczna-czy-wosk.html`
- `/poradnik/jak-przygotowac-auto-do-sprzedazy.html`
- `/poradnik/jak-dbac-o-tapicerke-samochodowa.html`

Każdy artykuł powinien zawierać:

1. jedno H1 odpowiadające na konkretne pytanie,
2. krótką odpowiedź na początku,
3. zdjęcia prawdziwej realizacji CarLux,
4. sekcję FAQ,
5. CTA do telefonu i formularza wyceny,
6. schema `Article`, `BreadcrumbList` i — jeśli występuje FAQ — `FAQPage`.

Nie należy tworzyć wielu krótkich podstron różniących się wyłącznie nazwą miejscowości.
